# Job Postings

Drop raw job descriptions here, one file per posting. Filename convention:

```
{company-slug}__{role-slug}__{YYYY-MM-DD}.md
```

Example: `acme__senior-product-manager__2026-04-22.md`

The file can contain the raw posting text — Markdown, plain text, or copy-pasted HTML is fine. The `/jd-extract` skill turns it into a structured analysis under `job-postings/<slug>/analysis.md`.
